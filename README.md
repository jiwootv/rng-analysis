# 난수의 난수성 (난수 알고리즘의 분포도)
## 탐구 요약
이 프로젝트는 컴퓨터가 생성한 수열이 얼마나 난수처럼 보이는지를 통계적·시각적으로 분석하는 탐구이다. 요즘 우리 생활에서는 난수를 사용하는 분야가 굉장히 많다. 그렇기에 현대로 올수록 난수의 질 등은 점점 중요해지고 있는데, 따라서 이 탐구에서는 카이제곱이나 3차원 2차원 분포를 통해 난수의 난수성에 대해서 알아볼 것이다.



## 난수 검정의 기준
1. 분포의 균등성 (카이제곱 검정 등 사용.)

**카이제곱 분포나 3차원, 2차원 분포도**를 통해 그래프를 그리고, 이를 통하여 **난수의 분포성과 균등성**을 판단한다. **즉 특정값이 몰렸는지 안 몰렸는지**를 확인하는 과정이라고 할 수 있다.


2. 평균과 분산의 적절성

평균이 너무 이론적 평균 근처이거나 표준편차등을 확인한다. (좀 1번이랑 겹침)

3. 독립성 (연속값 간 관계)

**기존의 데이터들을 통해 이후의 결과를 예측 가능한가**에 대한 내용이다. 일부 난수들 중 앞의 수열을 통해 예측 가능한 난수들이 있기에 넣어야 할 것 같아서 넣어 보았다. 예를 들어 몇개의 값을 보고 예측 알고리즘을 사용하여 그 난수 패턴을 알아낼 수 있을지와 같은 내용들. **2~3차원 산점도를 사용**하거나 뭐 알고리즘 있었는데(기억안남)을 사용해서 하지 않을까 생각해 본다.)


4. 같은 패턴이 다시 나타나는가
